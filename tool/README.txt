generator:
    A simple command line tool for create GRT-PELoader instance.

pe_loader:
    A simple command line tool for load exe/dll in memory.

http_server:
    A simple http file server for test instance with http server mode.
    It supported compression with gzip/deflate.

sc_loader:
    A mini shellcode loader for check resource leak.
    It will not load any DLL except Kernel32, KernelBase and ntdll.
